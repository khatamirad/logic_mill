# LogicMill Similarity Search App

## Overview
This app provides a Streamlit interface to perform similarity searches using the LogicMill API.

## Features
- Input API token
- Enter title and abstract
- Retrieve similar patents and publications
- View results in a table
- Download results as JSON and TXT

## Files
- `GUI_a_connected.py` – Streamlit GUI
- `similarity_search_json.py` – backend logic
